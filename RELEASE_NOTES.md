# v1.0.0-alpha: AI-Governed Substrate Registry

## 🛡️ Security-First Features

- **XCM Origin Guard**: Custom barrier (`RegistrationOrigin`) to block unauthorized cross-chain XCM messages.
- **Bounded Storage**: Implementation of `BoundedVec` for all user-generated inputs to prevent state bloat.
- **Hardware-Secured Signatures**: Integration with Google Cloud KMS for non-custodial, hardware-backed extrinsic signing.
